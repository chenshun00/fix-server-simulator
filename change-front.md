# FIX Server Simulator - 视觉风格改造文档

## 概述
将当前的FIX Server Simulator前端界面从基础Bootstrap样式升级为具有"专业、高可靠、科技感"的深色主题界面。

## 颜色方案
- 背景色 (Background): #0f172a (深蓝黑)
- 卡片/容器 (Surface): #1e293b (深灰蓝)
- 主文案 (Primary Text): #f8fafc (近白色)
- 强调/主色 (Accent): #3b82f6 (鲜亮蓝)

## 改造内容

### 1. 整体布局样式
- 将页面背景设置为 #0f172a
- 侧边栏背景设置为 #1e293b
- 主内容区域背景设置为 #1e293b
- 文字颜色设置为 #f8fafc

### 2. 侧边栏样式
- 背景: #1e293b
- 导航项文字: #f8fafc
- 激活状态背景: #3b82f6
- 激活状态文字: #ffffff
- 悬停效果: 轻微变亮效果

### 3. 表格样式
- 表头背景: #1e293b
- 表头文字: #f8fafc
- 表格行背景: #1e293b
- 表格行悬停背景: #334155
- 表格文字: #f8fafc
- 边框颜色: #334155

### 4. 按钮样式
- 主要按钮: 背景 #3b82f6，文字 #ffffff
- 次要按钮: 背景 #334155，文字 #f8fafc
- 悬停效果: 颜色加深或变亮
- 边框: 透明或同色系

### 5. 模态框样式
- 背景: #1e293b
- 文字: #f8fafc
- 输入框背景: #0f172a
- 输入框边框: #334155
- 输入框文字: #f8fafc

### 6. 卡片/面板样式
- 背景: #1e293b
- 边框: #334155
- 内容文字: #f8fafc

### 7. 状态指示器
- 连接状态: 绿色(#10b981)表示连接，红色(#ef4444)表示断开
- 其他状态指示使用强调色 #3b82f6

## 需要修改的文件
1. `/src/main/resources/static/css/app.css` - 主要样式文件
2. `/src/main/resources/static/index.html` - 可能需要微调HTML结构以适应新样式

## 实施步骤
1. 更新CSS文件，应用新的颜色方案
2. 调整表格、按钮、表单元素的样式
3. 优化模态框和其他组件的视觉效果
4. 确保所有元素在深色背景下具有良好的可读性
5. 测试各种组件在新主题下的显示效果
